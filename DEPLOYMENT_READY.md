## 🚀 Deploy Now - 3 Easy Steps

### ⚠️ Security Note
Before deploying, it's recommended to update Next.js for security patches:
```bash
npm install next@latest
npm run build
```
See `SECURITY_UPDATE_GUIDE.md` for details.

### Step 1: Push to GitHub

```bash
git add .
git commit -m "Portfolio ready for deployment"
git push origin main
```

### Step 2: Deploy to Vercel

1. Go to **[vercel.com](https://vercel.com)**
2. Click **"Sign Up"** (use GitHub)
3. Click **"New Project"**
4. Select your **portfolio** repository
5. Click **"Deploy"**

### Step 3: Wait 2-3 Minutes

That's it! Your portfolio will be live at:
`https://your-portfolio.vercel.app`

## 🎨 Customization Tips

### Update Content:

- **Education**: `components/Education.tsx`
- **Experience**: `components/Experience.tsx`
- **Projects**: `components/Portfolio.tsx`

### Change Colors:
- Edit `lib/constants.ts`
- Update `COLORS` object

### Add Projects:
1. Create new folder in `app/portfolio/`
2. Add `page.tsx` file
3. Update `projects-grid.tsx`

## 🆘 Troubleshooting

### Build Fails?
```bash
# Clear cache and rebuild
Remove-Item -Recurse -Force .next
npm run build
```

### Deployment Issues?
- Check Vercel dashboard for errors
- Verify all files are pushed to GitHub
- Ensure `package.json` has correct scripts

### Need Help?
- Check `DEPLOYMENT_GUIDE.md`
